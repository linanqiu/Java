Typecasting: When we want to put one datatype into another, it is called typecasting.
Eg: We have to convert bigger datatype into smaller datatype. Then typecasting is very useful

Static: If a function "xyz" is static in class "examplejava" we can call function like xyz.examplejava without 
calling object of examplejava class. Static member of class can be called directly using class name, without
using class object. Eg: parseInt function of Integer class is static.
